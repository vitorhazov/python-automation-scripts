# Python Automation Scripts

Repositório com vários scripts úteis de automação criados em Python.

### 📋 Scripts disponíveis
- Monitor de logs do FiveM com alertas no Discord
- Backup automático de bancos de dados MySQL
- Web scraper configurável com rotação de proxies
- Gerador e gerenciador de senhas fortes
- Auto sender de mensagens (Discord / Telegram)
- Limpeza automática de arquivos temporários
- Checker simples de usernames em redes sociais
- Monitor de ping e status de servidores

### 🛠️ Tecnologias utilizadas
- **Python 3.11+**
- `requests`, `selenium`, `schedule`, `telebot`, `colorama`, `mysql-connector`
- Suporte a arquivos `.env` para configurações

### Objetivo do projeto
Praticar Python avançado, criar automações úteis e integrar com FiveM, Discord e ferramentas de OSINT.
